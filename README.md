# Gain Live - Logo Assets

Official logo assets repository for Gain Live, a gaming & betting platform.

## Features

- **Logo Showcase**: Display all Gain Live logos in different styles and formats
- **Easy Download**: One-click download for each logo with proper file naming
- **File Size Display**: See the size of each logo before downloading
- **Organized Categories**: Logos grouped by type (White & Blue, Blue & White, Banner, Favicon, Black & White)
- **Responsive Design**: Works seamlessly on desktop and mobile devices

## Logo Categories

1. **White & Blue Logos** (4 variations)
2. **Blue & White Logos** (2 variations)
3. **Banner Logos** (2 variations)
4. **Favicon Logos** (2 variations)
5. **Black & White Logo** (1 variation)

## Usage

Simply open `index.html` in your browser to view all available logos and download the ones you need.
